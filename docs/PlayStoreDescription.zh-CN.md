# BT Input - Play Store 描述草案（中文）

## 应用简介（短描述）

把手机变成 Windows 蓝牙输入工具，离线直连，实时输入到电脑光标位置。

## 详细描述

BT Input 让你的手机秒变 Windows 输入设备：

- 使用你熟悉的系统输入法（拼音、语音、手写、第三方键盘）
- 通过蓝牙低功耗（BLE）直连电脑
- 文本实时发送到电脑当前输入光标
- 支持断线重连与同步恢复
- 支持常用特殊按键（Enter、Tab、方向键、Ctrl 组合）

无 Wi-Fi、无云同步、无互联网依赖，适合办公、会议、演示等场景。

## 主要功能

- BLE 离线直连
- 实时文本同步
- 智能差分发送，低延迟
- 长文本自动优化注入
- 连接状态可视化与调试统计

## 使用方式

1. 在 Windows 电脑上启动 BT Input 桌面端
2. 手机打开 BT Input，连接目标电脑
3. 在输入页输入文字，电脑端实时显示

## 隐私说明

BT Input 不收集云端数据，不上传你的输入内容。数据只在手机与电脑之间通过蓝牙本地传输。
